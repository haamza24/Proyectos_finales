Proyecto desarrollado en VHDL y ensamblador para el microcontrolador Picoblaze.
El proyecto consiste en 2 máquinas, un generador de numeros aletorios y un encriptador desarrollados en VHDL, se puede utilizar el generador aletorios para generar claves aletorios y poder encriptar
datos utilizando la maquina encriptadora, consiguiendo así, fusionar ambas máquinas.
